# Calculadora de Precios — Carpintería / Madera

Calculadora web para presupuestar trabajos en madera. Ingreso de materiales y cálculo automático de precios de venta. Archivo HTML único, sin dependencias.

## Funcionalidades

- Datos iniciales: costo de materiales, mano de obra, gastos generales
- Cálculo de precio final con margen de ganancia
- Interfaz simple y directa, lista para usar sin instalación

## Stack

- HTML5 + CSS3 + JavaScript vanilla (un solo archivo)
- Sin frameworks ni dependencias
